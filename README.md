# 🎯 SD DEAL - Service Marketplace

Complete Indian service booking platform with payments & reviews.

🚀 **[Live Demo](https://serene-germain-3.preview.emergentagent.com)**

## ✨ Features

- 🏠 14+ Indian service categories
- 💳 Stripe payment integration
- ⭐ Reviews & ratings
- 📱 Mobile responsive
- 💰 Provider wallet & withdrawals
- 💬 Real-time chat

## 🛠️ Tech Stack

**Frontend:** React 19, Tailwind CSS, Socket.io  
**Backend:** FastAPI, MongoDB, Stripe  
**Features:** JWT Auth, Payment escrow, Real-time chat
